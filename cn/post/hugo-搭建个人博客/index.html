Hugo 搭建个人博客

2023-11-171 min read (29 words)
#hugo 

安装 Hugo⌗ 

brew install hugo

建立网站⌗ 

hugo new site hugo

运行 Hugo 服务⌗ 

cd hugo/
hugo server

上传到 github.io⌗ 

cd blog/public
