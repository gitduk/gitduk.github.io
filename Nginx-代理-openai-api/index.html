安装 Nginx

ubuntu: sudo apt install nginx

配置 Openai 反向代理

在目录 /etc/nginx/conf.d 下创建文件 openai.conf

/etc/nginx/conf.d/openai.conf

重启 nginx 服务： sudo systemctl reload nginx
