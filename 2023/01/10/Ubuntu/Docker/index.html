docker run 参数

参数 作用
-d 让 Docker 在后台运行
-i 交互模式
-p/P 映射端口
-t 分配一个伪装的 tty
–name 命名 container
–network 为 container 分配网络
–dns 为 container 分配 DNS 服务器
-v 为 container 绑定数据卷
–rm container 存在时先移除，再创建
