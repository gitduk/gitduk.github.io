重定向

命令 说明
command > file 将输出重定向到 file
command < file 将输入重定向到 file
command >> file 将输出以追加的方式重定向到 file
<< tag 将开始标记 tag 和结束标记 tag 之间的内容作为输入

字体管理

Linux 字体安装和卸载很简单。

安装字体：直接把字体放到指定的目录，然后刷新字体缓存即可。
删除字体：只需从指定目录删除对应的字体文件，再刷新字体缓存。
Linux 字体放置目录有三个： 系统字体目录 /usr/share/fonts 用户字体目录

  • ~/.fonts
  • ~/.local/share/fonts

刷新字体缓存 fc-cache -fv
查看所有字体 fc-list | cut -d ':' -f2

/var/log/journal 过大问题

只保留近一周的日志
journalctl --vacuum-time=1w
只保留500MB的日志
journalctl --vacuum-size=500M
直接删除 /var/log/journal/ 目录下的日志文件
sudo rm -rf /var/log/journal/*
