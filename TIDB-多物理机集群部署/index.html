TIDB 多物理机集群部署

wukaige
2023-07-10
tidb

前情提要

买了三台服务器，内网 IP 如下：

  • 192.168.70.50
  • 192.168.70.51
  • 192.168.70.52

想搭建一个 TIDB 集群，只要有一台机器存活就能正常对外提供服务。

服务器环境准备

关闭系统 SWAP

禁用 THP

配置网络参数

配置 tidb 用户的打开文件句柄限制

将 CPU 频率调频模式从 powersave 设置为 performance

关闭防火墙

禁用 SELinux

mount disk with noatime

部署集群

编写集群拓扑文件: topology.yaml

检查并修复集群配置

部署集群

检查集群状态

开启必要端口并初始化集群

连接 TIDB 数据库并修改密码
