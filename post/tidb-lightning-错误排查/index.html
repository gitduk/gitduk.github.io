TiDB-Lightning 错误排查

Keywords: #TiDB #Lightning

raft entry is too large

修改 TiDB 配置

tiup cluster edit-config tidb

  tikv:
    raftstore.raft-entry-max-siz: 128MB

重启 tikv 节点

tiup cluster reload tidb-cluster -R tikv
